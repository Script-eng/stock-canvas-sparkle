# NSE Stock Market Dashboard

A modern, real-time dashboard for the Nairobi Securities Exchange with ML-powered stock predictions.

## Live System

- **Production URL:** [https://nse.softwarepulses.com](https://nse.softwarepulses.com)
- **Frontend Repository:** [https://github.com/Script-eng/stock-canvas-sparkle.git](https://github.com/Script-eng/stock-canvas-sparkle.git)
- **Backend Repository:** [https://github.com/Script-eng/thesis-ml.git](https://github.com/Script-eng/thesis-ml.git)

## Features

- Real-time market data with 5-second updates
- ML predictions (LSTM + Prophet ensemble)
- Interactive stock detail modals
- Market indices tracking
- Historical charts and analytics
- Responsive design for mobile and desktop

**Running the project**

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone 

# Step 2: Navigate to the project directory.
cd 

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```



## What technologies are used for this project?

This project is built with:

- **React 18** - Modern UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool
- **Tailwind CSS** - Utility-first styling
- **Shadcn/ui** - High-quality component library
- **Recharts** - Interactive data visualizations
